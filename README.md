# 谁是卧底

网页版聚会推理游戏，一台手机传着玩。

**在线体验：** https://colorzc-xjtu.github.io/who-is-spy/

## 功能

- 4-12 人游戏，支持卧底 + 白板角色
- 50+ 内置词对，7 个分类（食物、动物、日常、职业、影视、地点、抽象）
- 支持自定义词语
- 翻牌查看身份，防偷看
- 讨论计时器、玩家淘汰标记
- 深色 / 浅色主题
- 手机端自适应，扫码即玩

## 游戏规则

1. 设置玩家人数、卧底数、白板数，选择词语分类
2. 每人轮流点击卡牌查看自己的身份和词语
3. 所有人查看完毕后，轮流用一句话描述自己的词（不能直接说）
4. 每轮讨论后投票淘汰最可疑的人
5. 平民找出所有卧底即胜利；卧底存活到最后即胜利

## 技术栈

纯 HTML / CSS / JS，零依赖，无需构建，GitHub Pages 直接部署。
